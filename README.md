## Installation

In a browser:
```html
<script type="text/javascript" src="_z.js"></script>
```

Using npm:
```shell
$ npm i @mphpmaster/underz
```
Note: add --save if you are using npm < 5.0.0

Using composer:
```shell
$ composer require mphpmaster/underz
```
